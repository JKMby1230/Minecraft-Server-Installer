* **三核心支援**：Fabric / Forge / NeoForge 自動下載安裝。

### ☕ Java 版本需求 (重要)
本軟體會自動偵測，但請確保您的電腦已安裝對應的 Java：

| Minecraft 版本 | 需要的 Java 版本 |
| :--- | :--- |
| **1.20.5 - 1.21+** | **Java 21** |
| **1.18 - 1.20.4** | **Java 17** |
| **1.16.5 以下** | **Java 8** |

### 📖 使用教學
1.  **下載與解壓**：下載上方的 `.zip` 檔並解壓縮。
2.  **執行軟體**：打開 `MinecraftServerInstaller.exe`。
3.  **安裝設定**：
    * 選擇安裝路徑 (空資料夾)。
    * 選擇核心與版本。
    * 點擊 **「開始安裝」**。
4.  **啟動伺服器**：進入安裝好的資料夾，點擊 `start.bat` 即可開服！
