# 🛠️ Minecraft Server Installer
## 最簡單的 Minecraft 伺服器架設工具

歡迎使用 **Minecraft Server Installer**！這是一個專為玩家設計的免費開源工具，讓你在 3 分鐘內架設好 Fabric 或 Forge 伺服器。

[👉 下載最新版本 (v0.0.1)](你的GitHub_Releases下載連結)

---

### ✨ 軟體特色
* **雙核心支援**：自動下載 Fabric / Forge。
* **版本齊全**：支援 1.20.4 至 1.12.2 等歷史版本。
* **一鍵設定**：圖形化介面調整記憶體、Port、白名單。
* **網路工具**：內建 IP 查詢功能 (LAN/Public)。

---

### 📖 使用教學
1.  下載 `.exe` 檔案。
2.  選擇安裝資料夾與版本。
3.  點擊「開始安裝」。
4.  啟動 `start.bat` 即可遊玩！
