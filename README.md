Minecraft Server Installer v0.0.20
📖 簡介
這是一個專為 Minecraft 玩家設計的伺服器快速安裝工具。v0.0.20 版本進一步優化了自動化流程，讓架設過程更加直覺且穩定。

🚀 v0.0.20 更新亮點
✨ 新增功能
架構優化：重構了後端下載邏輯，提升伺服器核心下載速度。

介面調整：優化了控制台輸出訊息，更容易掌握安裝進度。

IP 自動檢測：新增自動獲取區網 (LAN) 與公網 IP 的功能，方便玩家連線。

🐞 修復項目
修正了在某些 Windows 版本下路徑包含空白字元導致的安裝失敗問題。

修復了 Java 版本偵測邏輯的錯誤。

⚠️ 重要變動
移除第三方穿透：此版本起不再支援 Playit.gg 與 ngrok，專注於 LAN 與 Public IP 連線模式，以確保連線品質與安全性。

🛠 安裝與使用
系統需求
操作系統：Windows 10/11

環境：Python 3.x (若執行源碼版本)

網路：穩定的網路連線

使用步驟
前往 Releases 下載最新的 v0.0.20 壓縮包。

解壓縮後執行 MinecraftServerInstaller.exe (或運行 Python 腳本)。

按照畫面指示選擇伺服器類型與版本。

完成安裝後，透過輸出的公網 IP 即可讓好友加入。
